End-to-end flow
  1. Admin configures Entra + ClassLink in DFC Org Settings
  2. Admin sets SSO Required + Rostering in Policy
  3. DFC sync job pulls ClassLink roster → local DB
  4. Teacher signs in → classes loaded from DFC DB
  5. Teacher selects class → starts session
  6. Student signs in → DFC resolves enrolled class → auto-joins
